# 文章高亮功能使用指南

本文档站点集成了强大的文章高亮功能，包括选择高亮、搜索高亮和清除高亮。

## 功能概览

### 1. 高亮选中文本
- **位置**: ArticleControl 悬浮窗 / 右键菜单
- **图标**: 🖍️ 高亮笔图标
- **使用方法**:
  1. 选中文章中的任意文本
  2. 点击 ArticleControl 中的"高亮选中"按钮，或
  3. 右键点击选中的文本，选择"高亮选中"
- **效果**: 选中的文本会被黄色背景高亮显示

### 2. 搜索高亮（全局查找）
- **位置**: ArticleControl 悬浮窗 / 右键菜单
- **图标**: 🔍 搜索图标
- **使用方法**:
  1. 选中要搜索的文本（例如一个关键词）
  2. 点击 ArticleControl 中的"搜索高亮"按钮，或
  3. 右键点击选中的文本，选择"搜索高亮"
- **效果**: 
  - 文章中所有匹配的文本都会被高亮显示
  - 会显示找到的匹配项数量

### 3. 清除高亮
- **位置**: ArticleControl 悬浮窗 / 右键菜单
- **图标**: ❌ 清除图标
- **使用方法**:
  1. 点击 ArticleControl 中的"清除高亮"按钮，或
  2. 右键点击文章任意位置，选择"清除高亮"
- **效果**: 移除所有高亮标记，恢复原始文本显示

## 使用场景

### 场景 1: 学习重点内容
在阅读技术文档时，可以高亮重要的概念、API 名称或关键代码片段，方便后续快速定位。

### 场景 2: 查找关键词
想要了解某个术语在文档中出现的所有位置？选中该术语，使用"搜索高亮"功能，即可看到所有出现的地方。

### 场景 3: 对比学习
高亮多个相关概念，对比它们在文档中的使用场景和上下文。

## 技术实现

- 使用 `TreeWalker` API 遍历 DOM 文本节点
- 使用 `<mark>` 元素进行高亮标记
- 支持重叠高亮和多次高亮
- 自动处理文本节点的分割和合并

## 注意事项

1. 高亮功能只在当前页面生效，刷新页面后高亮会消失
2. 切换到其他文档页面时，高亮会自动清除
3. 高亮不会影响文档的原始内容，只是视觉上的标记
4. 搜索高亮是大小写敏感的精确匹配
