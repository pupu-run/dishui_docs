# Markdown 简介

## 什么是 Markdown？

Markdown 是一种轻量级标记语言，由 John Gruber 在 2004 年创建。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。

## 为什么使用 Markdown？

### 优点

- **简单易学**：语法简洁，几分钟即可上手
- **纯文本**：可以使用任何文本编辑器编辑
- **跨平台**：在任何操作系统上都能使用
- **版本控制友好**：纯文本格式便于 Git 等版本控制系统管理
- **专注内容**：让你专注于写作内容而非格式
- **广泛支持**：GitHub、Reddit、Stack Overflow 等平台都支持

### 常见应用场景

1. **技术文档**：API 文档、用户手册、开发指南
2. **博客文章**：许多静态网站生成器使用 Markdown
3. **README 文件**：项目说明文档
4. **笔记记录**：个人知识管理
5. **在线协作**：团队文档、Wiki
6. **电子书**：可转换为 PDF、ePub 等格式

## Markdown 的历史

- **2004 年**：John Gruber 和 Aaron Swartz 创建了 Markdown
- **2012 年**：CommonMark 项目启动，旨在标准化 Markdown
- **现在**：各种 Markdown 变体（GitHub Flavored Markdown、MultiMarkdown 等）
