# 基础语法

## 标题（Headings）

Markdown 支持六级标题，使用 `#` 符号表示。`#` 的数量代表标题级别。

### 语法

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 效果

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

### 注意事项

- `#` 和标题文字之间要有一个空格
- 标题前后最好空一行，提高可读性
- 一个文档通常只有一个一级标题

### 另一种标题语法（不推荐）

```markdown
一级标题
========

二级标题
--------
```

## 段落（Paragraphs）

段落之间用一个或多个空行分隔。

### 语法

```markdown
这是第一段文字。

这是第二段文字。


这是第三段文字（前面有多个空行也只显示为一个空行）。
```

### 效果

这是第一段文字。

这是第二段文字。


这是第三段文字（前面有多个空行也只显示为一个空行）。

## 换行（Line Breaks）

在 Markdown 中，单个回车不会产生换行。要创建换行，需要：

### 方法一：行末添加两个空格

```markdown
第一行（后面有两个空格）  
第二行
```

效果：
第一行（后面有两个空格）  
第二行

### 方法二：使用 HTML 标签

```markdown
第一行<br>
第二行
```

效果：
第一行<br>
第二行

### 方法三：空一行（创建新段落）

```markdown
第一段

第二段
```

## 强调（Emphasis）

### 斜体（Italic）

使用单个 `*` 或 `_` 包围文字。

```markdown
*这是斜体*
_这也是斜体_
```

效果：*这是斜体* _这也是斜体_

### 粗体（Bold）

使用两个 `*` 或 `_` 包围文字。

```markdown
**这是粗体**
__这也是粗体__
```

效果：**这是粗体** __这也是粗体__

### 粗斜体（Bold and Italic）

使用三个 `*` 或 `_` 包围文字。

```markdown
***这是粗斜体***
___这也是粗斜体___
```

效果：***这是粗斜体*** ___这也是粗斜体___

### 删除线（Strikethrough）

使用两个 `~` 包围文字（GFM 扩展语法）。

```markdown
~~这是删除线~~
```

效果：~~这是删除线~~

## 水平线（Horizontal Rules）

使用三个或更多的 `-`、`*` 或 `_` 创建水平线。

### 语法

```markdown
---

***

___

- - -

* * *
```

### 效果

---

***

___

## 实践练习

2. 几个段落，包含粗体、斜体和删除线
3. 使用水平线分隔不同的部分

